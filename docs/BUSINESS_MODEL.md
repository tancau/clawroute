## ⚠️ 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| Key 失效 | 实时验证 + 自动切换 + 用户通知 |
| 法律风险 | 只允许特定平台 + 用户协议 |
| 资源池枯竭 | 要求用户贡献 2-3 个 Key |
| 成本失控 | 免费用户限流 + 付费优先 |
| 恶意注册 | 手机验证 + 设备指纹 + 行为分析 |
