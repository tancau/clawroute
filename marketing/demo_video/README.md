## 📁 目录结构

```
demo_video/
├── README.md                 # 本文件（索引）
```

## 📋 文件说明

### 1. 视频脚本 (`demo_script.md`)

三个版本的视频脚本：
- **30秒版**：快速展示核心价值
- **60秒版**：完整展示功能与优势
- **2分钟版**：深度讲解技术细节

包含：分镜描述、配音稿、时间戳、字幕建议

### 2. 录制指南 (`recording_guide.md`)

完整的视频录制教程：
- 录屏工具选择与配置
- 浏览器设置
- 鼠标移动技巧
- 配音录制方法
- 导出格式建议

### 3. 背景音乐 (`music_suggestions.md`)

免版权 BGM 推荐：
- 适合开发者工具的音乐风格
- 免费音乐平台列表
- 具体曲目推荐
- 音量混音建议

### 4. 封面设计 (`thumbnail_ideas.md`)

视频封面设计指南：
- 品牌色系定义
- 三种设计方案
- 文字排版建议
- 平台尺寸适配

### 5. 流程图 (`flow_diagrams/`)

Mermaid 格式的可视化素材：
- `request_flow.mmd` - 智能路由流程
- `cost_comparison.mmd` - 成本对比
- `architecture.mmd` - 系统架构

**渲染方式**：
- 在线：[Mermaid Live Editor](https://mermaid.live)
- VS Code：安装 Mermaid 插件
- 导出：PNG / SVG 格式

## 🚀 快速开始

### 第一步：准备素材

1. 访问 https://hopllm.com 了解产品
2. 准备 HopLLM Logo（SVG/PNG）
3. 选择 BGM（参考 `music_suggestions.md`）

### 第二步：制作流程图

```bash
# 使用 Mermaid Live Editor 渲染
https://mermaid.live

# 或使用 VS Code 插件
# 安装 "Markdown Preview Mermaid Support"
```

### 第三步：录制视频

1. 参考 `recording_guide.md` 设置环境
2. 使用 `demo_script.md` 的脚本录制
3. 后期剪辑、添加配音、混音

### 第四步：制作封面

1. 参考 `thumbnail_ideas.md` 设计封面
2. 使用 Figma / Canva 快速制作
3. 导出 1920x1080 PNG 格式

### 第五步：发布

- YouTube / B站：1080p H.264
- 微信视频号：720p 竖版
- 社交媒体：方形封面
