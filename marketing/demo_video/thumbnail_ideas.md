# HopLLM 视频封面设计建议

## 一、品牌色系

### 主色调
| 颜色 | HEX | RGB | 用途 |
|------|-----|-----|------|
| 品牌紫 | #4F46E5 | 79, 70, 229 | 主视觉、Logo、CTA |
| 深紫 | #7C3AED | 124, 58, 237 | 渐变、强调 |
| 亮紫 | #A855F7 | 168, 85, 247 | 高光、装饰 |

### 辅助色
| 颜色 | HEX | RGB | 用途 |
|------|-----|-----|------|
| 深蓝灰 | #0F172A | 15, 23, 42 | 背景 |
| 浅灰 | #F8FAFC | 248, 250, 252 | 文字、卡片 |
| 绿色 | #10B981 | 16, 185, 129 | 节省、成功 |
| 红色 | #EF4444 | 239, 68, 68 | 对比、警告 |

### 渐变推荐
```css
/* 主渐变 */
background: linear-gradient(135deg, #4F46E5 0%, #7C3AED 50%, #A855F7 100%);

/* 深色背景渐变 */
background: linear-gradient(180deg, #0F172A 0%, #1E1B4B 100%);

/* 玻璃效果 */
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(10px);
```

## 二、封面设计方案

### 方案 A：成本对比型（推荐）

**视觉元素：**
```
┌──────────────────────────────────────────────┐
│  [深蓝灰渐变背景]                              │
│                                              │
│      HopLLM Logo                             │
│      ┌─────────┐    ┌─────────┐             │
│      │ $500    │ ─→ │ $17.5   │             │
│      │ 原成本   │    │ 节省 97% │             │
│      └─────────┘    └─────────┘             │
│                                              │
│      "智能路由，自动省钱"                      │
│      hopllm.com                              │
│                                              │
└──────────────────────────────────────────────┘
```

**设计要点：**
- 左侧红色/灰色框：原成本 $500
- 右侧绿色框：节省后 $17.5 + 大大的 "97%↓"
- 中间箭头：品牌紫色渐变
- Logo 在顶部居中
- Slogan 在底部

### 方案 B：路由可视化型

**视觉元素：**
```
┌──────────────────────────────────────────────┐
│  [深色背景 + 网格线装饰]                       │
│                                              │
│      用户请求 ──→ [HopLLM] ──→ 响应          │
│                      │                       │
│         ┌────────────┼────────────┐          │
│         ↓            ↓            ↓          │
│     [GPT-4]     [Claude]    [GPT-3.5]       │
│                                              │
│      "一个请求，自动选最优模型"                │
│                                              │
└──────────────────────────────────────────────┘
```

**设计要点：**
- 中心 HopLLM Logo 突出
- 分支线条用品牌紫色
- 模型图标小而清晰
- 简洁的流程图风格

## 三、文字排版建议

### 标题层级

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 主标题 | 72px | Bold (700) | #F8FAFC |
| 副标题 | 48px | Medium (500) | #94A3B8 |
| Slogan | 36px | Regular (400) | #A855F7 |
| 价格数字 | 96px | Black (900) | #10B981 (绿色) 或 #EF4444 (红色) |

### 字体推荐

**中文：**
- 思源黑体 (Noto Sans SC)
- 阿里巴巴普惠体
- 站酷高端黑（标题）

**英文：**
- Inter（UI）
- SF Pro Display（Apple 风格）
- Manrope（科技感）

**数字：**
- JetBrains Mono（等宽）
- Fira Code（代码感）
