## 二、录制环境设置

### 浏览器设置

```bash
# 推荐窗口尺寸
1920 x 1080 (16:9 标准)

# 或使用 Chrome 无 UI 元素模式
1. 按 F11 进入全屏（可选）
2. 隐藏书签栏：Ctrl+Shift+B (Windows) / Cmd+Shift+B (Mac)
3. 关闭浏览器扩展图标（减少干扰）
4. 使用干净的无痕窗口（避免个人信息）
```

### 桌面准备

```
✅ 隐藏桌面图标
✅ 使用纯色/简洁壁纸（深色 #0F172A 或浅色 #F8FAFC）
✅ 关闭通知（勿扰模式）
✅ 清理任务栏/Dock
✅ 关闭无关应用
```

### OBS 推荐设置

```
视频设置：
- 基础分辨率：1920 x 1080
- 输出分辨率：1920 x 1080
- FPS：30（演示足够）或 60（流畅动画）
- 格式：MP4

音频设置：
- 采样率：48 kHz
- 声道：立体声
- 比特率：192 kbps

编码器：
- NVIDIA GPU：NVENC
- AMD GPU：AMF
- Intel GPU：QSV
- CPU：x264（无 GPU 时）
```

## 三、鼠标移动技巧

### 基本原则

1. **平滑移动**：避免快速抖动，匀速移动鼠标
2. **停顿强调**：点击前稍作停顿，让观众看清位置
3. **放大关键区域**：使用 OBS 缩放功能突出细节
4. **避免画圈**：不要无意义地移动鼠标

### 移动轨迹建议

```
场景 1：代码演示
┌─────────────────────────────────────┐
│  1. 先停在 base_url 上 (2s)         │
│  2. 平滑移动到 model: "auto" (1.5s) │
│  3. 点击运行按钮 (停顿 1s)           │
│  4. 等待响应展示 (3s)                │
└─────────────────────────────────────┘

场景 2：成本计算器
┌─────────────────────────────────────┐
│  1. 输入调用次数 (停顿显示)          │
│  2. 移动到 Token 输入框              │
│  3. 平滑滚动到结果区域               │
│  4. 高亮节省金额 (2s)                │
└─────────────────────────────────────┘
```

### 鼠标光标建议

- **使用高亮光标**：BigBluePointer、Cursor Pro
- **添加点击效果**：OBS 鼠标点击特效插件
- **颜色对比**：红色/黄色光标在深色背景更醒目

## 四、配音录制技巧

### 录音环境

```
✅ 安静房间（关闭空调、风扇）
✅ 软装环境（减少回声）
✅ 使用防喷罩
✅ 耳机监听（避免喷麦）
```

### 配音风格

| 类型 | 说明 |
|------|------|
| **语速** | 中等偏快（150-170 字/分钟） |
| **语气** | 专业但不生硬，带亲和力 |
| **停顿** | 句子之间 0.5s，段落之间 1s |
| **重音** | 落在关键词：节省、智能、自动、零迁移 |

### 录制流程

```
1. 录制屏幕 → 不配音（先拿画面）
2. 看画面配音 → 单独录音轨道
3. 后期合成 → 对齐音画
```

## 五、导出设置

### 视频格式

```
推荐设置：
- 格式：MP4 (H.264)
- 分辨率：1920 x 1080 (1080p)
- 帧率：30 fps
- 比特率：8-12 Mbps（YouTube 推荐）
- 音频：AAC, 192 kbps

上传优化：
- YouTube：1080p, H.264, 8-12 Mbps
- B站：1080p, H.264, 6-8 Mbps
- 微信视频号：720p, H.264, 4-6 Mbps
- Twitter：1280 x 720, H.264, 5 Mbps
```

### 文件命名

```

HopLLM_Demo_60s_v1.0.mp4
```

## 七、快速录制命令（OBS）

```bash
# 启动 OBS 录制（需先配置场景）
obs --startrecording

# 或使用命令行录制（Linux/Mac）
ffmpeg -f x11grab -video_size 1920x1080 -i :0.0 \
  -f alsa -i default -c:v libx264 -preset ultrafast \
  -c:a aac output.mp4
```
