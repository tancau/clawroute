## 📝 详细描述

### Hook
**Stop overpaying for AI API calls.** Every request gets the best model for the job — automatically.

### Product Description

HopLLM is a smart routing API proxy that helps developers save 60-80% on AI API costs without sacrificing quality. Simply replace your OpenAI base_url, use `model: "auto"`, and let HopLLM handle the rest.

**How it works:**
1. **Intent Classification** — Analyzes each request to understand what's being asked
2. **Smart Model Selection** — Routes to the optimal model based on complexity, context, and task type
3. **Automatic Fallback** — Handles rate limits and failures with seamless failover

### Key Features

✅ **Zero Migration Cost** — OpenAI-compatible API, change one line of code
✅ **Intelligent Routing** — 3-layer intent classification + multi-factor model scoring
✅ **Stream Support** — SSE streaming for real-time responses
✅ **Auto Retry & Failover** — Handles 429/5xx errors automatically
✅ **Precise Billing** — Pay per token × model price, no fixed fees
✅ **Key Pool** — Load balancing across multiple API keys
